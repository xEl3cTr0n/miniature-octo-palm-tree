# Roadmap

## Phase 1: Backend MVP

- Case upload API
- Typed evidence schemas
- Deterministic retrieval
- Contradiction checks
- Missing-evidence checks
- Report endpoint

## Phase 1.5: Real Data And Workflow Baseline

- NHTSA vehicle complaint and recall adapter
- Scripted live-data demo
- FastAPI endpoint for public data source querying
- Stored case workspace endpoints
- Reviewer report endpoint for stored cases
- Deterministic evaluation dataset and CLI/API harness
- Clean CI with deterministic unit tests

## Phase 2: Multimodal AI

- OCR and layout-aware document parsing
- ASR for claimant calls
- Image captioning and object detection
- Image embeddings for visual retrieval
- Cross-encoder reranking

## Phase 3: Agentic workflow

- LangGraph ingestion agent
- Retrieval agent
- Policy comparison agent
- Contradiction agent
- Evaluation agent
- Report agent

## Phase 4: Portfolio polish

- Next.js dashboard
- Evidence graph visualization
- Eval dashboard
- Demo video
- Deployed public demo with synthetic data
